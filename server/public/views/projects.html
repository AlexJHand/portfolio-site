LFC-Scheduler

- Link - GitHub Link

This application scrapes the web for Liverpool Football Club game times and then converts and displays those times in Central (and other American) time zone, as well as various other data about the current Premier League season.

Technologies Used

React, Node, Express, Request, Cheerio, HTML5, and CSS3.

Social Drinker

- Link - GitHub Link

[Prime Solo Project] An application to help users remember and track which beers they have consumed, as well as their opinions about them. Social Drinker allows them to search by beer name or brewery, and then lets them input a rating as well as a comment.

Technologies Used

HTML5, CSS3, AngularJS, NodeJS, Express, Passport, PostgreSQL, BreweryDB api, Sweet Alerts, Google Fonts.

Minnesota Missing Children

- GitHub Link

[Prime Group Project] An application built for Missing Children Minnesota, to allow case workers to fill out an intake form for a new case, which will then be able to be printed to a pdf. In addition to this, a portion of the data from the form will be stored in a database for reporting purposes.

Technologies Used

HTML5, CSS3, AngularJS, NodeJS, Express, Passport, PostgreSQL, Bootstrap, Angular Materials, Sweet Alerts, and ChartsJS.
